# AI Resume Scorer 🚀

An AI-powered resume analysis platform that helps users improve their chances of getting shortlisted by analyzing their resume against a specific job description.

Users can upload their resume, paste a job description, and the AI evaluates the resume by providing a score, identifying missing skills, and suggesting improvements to make the resume more relevant for the target role.

## ✨ Features

- Upload resume for AI analysis
- Paste job description for comparison
- AI-powered resume scoring
- Match resume skills with job requirements
- Identify missing keywords and skills
- Get personalized resume improvement suggestions
- User-friendly and responsive interface

## 🔄 How It Works

1. User uploads their resume
2. User enters the job description they want to apply for
3. AI analyzes the resume based on the job requirements
4. The system generates:
   - Resume score
   - Matching skills
   - Missing keywords
   - Improvement suggestions
